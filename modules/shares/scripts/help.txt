Shares (cap table management):
  shares table                Show cap table with percentages and vesting
  shares export               Output cap table as CSV
  shares holders              List all shareholders with totals
  shares history [holder]     Show share events (optionally filter by holder)
  shares check                Validate ledger consistency
  shares brief                Compact context dump for agent warm-up
  shares pdf table            Cap table as PDF
  shares pdf history          Share event history as PDF
  shares pdf holder <id>      Individual holder statement as PDF
